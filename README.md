# Onepage.io

This repository hosts a collection of single-file HTML5 games. Each game lives in its own HTML file under the `games/` directory and includes all CSS and JavaScript inline for easy hosting.

Available games:

- Tic‑Tac‑Toe — `games/tic-tac-toe.html` (two-player hotseat)
- Snake — `games/snake.html`
- Minesweeper — `games/minesweeper.html`
- Cultivation RPG — `games/cultivation-rpg.html`
- Snake — `games/snake.html`
- Minesweeper — `games/minesweeper.html`
- Cultivation RPG — `games/cultivation-rpg.html`

How to run locally:

1. Start a simple static server from the repository root (Python 3):

```bash
python3 -m http.server 8000
```

2. Open your browser and go to `http://localhost:8000/index.html`.

See `copilot-instructions.md` for contribution guidelines and single-file requirements.
